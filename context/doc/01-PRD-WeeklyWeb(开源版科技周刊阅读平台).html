Weekly Web 产品需求文档（PRD）V1.0

1. 文档信息

项目名称 Weekly Web
项目类型 开源项目
产品定位 科技爱好者周刊 Web 阅读平台
内容来源 ruanyf/weekly
版本 V1.0

2. 产品背景

科技爱好者周刊长期以 Markdown 形式维护于 GitHub 仓库中， 虽然内容质量较高，但在实际阅读过程中存在阅读体验差、检索困难、 内容沉淀能力弱等问题。

用户在阅读长篇周刊时，经常需要打开大量外部链接， 返回后难以恢复原阅读位置； 同时缺乏全文检索能力，无法快速定位历史内容。

因此需要构建独立 Web 平台， 提升内容消费效率和阅读体验。

3. 产品目标

3.1 业务目标

  • 提高周刊内容可读性
  • 提高历史内容检索效率
  • 提高用户持续阅读体验
  • 构建长期维护的开源项目

3.2 用户目标

  • 快速查找技术内容
  • 快速定位历史周刊
  • 阅读不中断
  • 收藏有价值内容

4. 用户分析

目标用户

  • 程序员
  • 技术爱好者
  • 产品经理
  • AI 从业者
  • 互联网从业者

用户特征

  • 具备技术阅读习惯
  • 关注行业趋势
  • 经常搜索历史内容
  • 重度桌面端用户
  • 兼顾移动端阅读场景

5. 用户故事

US-01 阅读周刊

作为用户， 我希望浏览历史周刊， 从而快速找到感兴趣的内容。

US-02 全文搜索

作为用户， 我希望搜索技术关键词， 从而快速定位相关内容。

US-03 阅读恢复

作为用户， 我希望再次打开周刊时恢复上次阅读位置， 从而避免重复查找。

US-04 收藏内容

作为用户， 我希望收藏有价值内容， 从而方便后续查阅。

6. 功能需求

FR-01 首页

功能描述

  • 展示周刊列表
  • 按发布时间倒序展示
  • 支持年份筛选
  • 提供搜索入口
  • 展示最新一期周刊

验收标准

  • 用户可浏览全部周刊
  • 用户可按年份筛选
  • 用户可进入详情页

FR-02 周刊详情页

功能描述

  • Markdown 渲染
  • 图片展示
  • 代码高亮
  • 目录导航
  • 章节定位
  • 章节折叠
  • 响应式布局

业务规则

  • 自动生成目录
  • 支持目录跳转
  • 支持移动端阅读

FR-03 阅读进度

功能描述

  • 自动记录阅读位置
  • 自动恢复阅读位置
  • 展示阅读进度

业务规则

  • 游客保存至本地
  • 登录用户同步服务器

FR-04 外链访问

功能描述

  • 所有外链默认新标签页打开

业务规则

  • 不离开当前阅读页面
  • 不影响阅读进度

FR-05 全文搜索

功能描述

  • 支持全文搜索
  • 支持关键词高亮
  • 支持结果定位

搜索结果展示

  • 期数
  • 章节名称
  • 匹配片段
  • 关键词高亮

业务规则

  • 点击结果直接跳转命中位置
  • 自动滚动至匹配内容

FR-06 收藏系统

访问权限

登录用户

收藏对象

  • 整期周刊
  • 具体条目

功能描述

  • 收藏
  • 取消收藏
  • 查看收藏列表

FR-07 阅读历史

访问权限

登录用户

功能描述

  • 查看最近阅读
  • 查看阅读进度
  • 继续阅读

FR-08 主题模式

访问权限

所有用户

支持模式

  • 浅色模式
  • 深色模式
  • 跟随系统

业务规则

  • 游客保存本地配置
  • 登录用户同步服务器配置

FR-09 登录

访问权限

所有用户

功能描述

  • 支持 GitHub 登录
  • 支持 Linuxdo 登录

业务规则

  • 收藏、阅读历史、跨设备同步需要登录
  • V1 仅支持 GitHub 和 Linuxdo 登录

7. 页面结构

首页
├── 周刊列表
├── 搜索入口
│
├── 周刊详情
│   ├── 目录导航
│   ├── 阅读内容
│   ├── 收藏
│   └── 阅读进度
│
├── 搜索页
│
└── 用户中心
    ├── 收藏列表
    ├── 阅读历史
    └── 个人设置

8. 非功能需求

性能要求

  • 首页加载时间 ≤ 2秒
  • 搜索响应时间 ≤ 300毫秒
  • 移动端流畅滚动

兼容性要求

  • Chrome
  • Edge
  • Safari
  • Firefox

响应式要求

  • 桌面端
  • 平板端
  • 移动端

9. 需求边界

V1 不包含

  • 评论系统
  • RSS订阅
  • 纠错反馈
  • AI问答
  • 内容推荐
  • 会员体系
  • 付费功能

10. 验收标准

  • 用户可正常浏览全部周刊
  • 用户可通过关键词搜索内容
  • 搜索结果支持定位
  • 阅读位置可自动恢复
  • 登录用户可收藏内容
  • 登录用户可查看阅读历史
  • 支持深色模式、浅色模式、跟随系统模式
  • 支持桌面端与移动端访问
