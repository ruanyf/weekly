# Security Policy / 安全策略

## Malicious Link Reporting / 恶意链接反馈

This weekly newsletter shares project recommendations and website links. If you identify a recommended link that has been compromised, contains malware, or poses a security risk, please report it immediately.
- Please do **not** open a public issue.
- Report the concern privately via GitHub's private vulnerability reporting feature or by emailing the project maintainer directly.

本周刊推荐各类开源项目和网络链接。如果您发现推荐的项目或链接失效、被篡改或存在安全风险，请勿通过公开 Issue 报告，请使用 GitHub 的私有漏洞报告功能或直接发送电子邮件联系维护者。
